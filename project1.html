STUDENT PERFORMANCE PREDICTION AND ANALYSIS

Description: This project aims to predict and analyze student performance using a machine learning workflow. The process includes data collection, cleaning, exploratory data analysis (EDA), feature engineering, model training, evaluation, and deployment. The goal is to provide actionable insights into student performance and suggest interventions to improve outcomes.

(deploye the model)
Technologies Used:
  • Python 3.9: Used for data preprocessing, EDA, and machine learning model development.
  • NumPy and Pandas: For handling and analyzing structured data efficiently.
  • Seaborn and Matplotlib: For visualizing patterns and trends in the dataset.
  • Scikit-learn: For training and evaluating machine learning models.
  • Flask: A lightweight web framework used to deploy the machine learning model as an API.
  • HTML, CSS, and JavaScript: To create an interactive and user-friendly web interface for the application.
